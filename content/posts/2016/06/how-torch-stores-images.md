```
require('image')
imgPath = "image.jpg"
img = torch.Tensor(1, 3, imgDim, imgDim)
img[1] = image.load(imgPath, 3, byte)
```
image is stored as float, conversion is (intensity/255.). stored top->bottom, line by line. format RGB.
```
display
require 'gnuplot'
gnuplot.figure(1)
gnuplot.imagesc(img[1])
```
