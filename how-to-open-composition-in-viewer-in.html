How to open composition in viewer in Adobe After Effects using scripts

Date Thu 24 October 2013 Tags after effects (aae) / scripting
// Make a composition var comp = app.project.items.addComp('MyComp', 1920, 1080, 1.0, 10, 25.0 ); // Open it in viewer comp.openInViewer(); This solution works only in AAE CS 6.0 My version of some cross-platform code function OpenInViewer( comp ) {     var version = app.version.match(/(\d+\.\d+).*/)[1];     if( version >= 11.0 )         comp.openInViewer() ;     else     {         var duration = comp.workAreaDuration;         comp.workAreaDuration = 2*comp.frameDuration;         comp.ramPreviewTest("",1,"");         comp.workAreaDuration = duration;     } }
