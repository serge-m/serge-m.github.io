Mount yandex webdav on local dir

Date 2016-07-16T13:45:00, Sat Tags useful / linux 
    apt-get install davfs2
    mkdir /mnt/yandex.disk
    mount -t davfs https://webdav.yandex.ru /mnt/yandex.disk/

    # check: 
    df -h /mnt/yandex.disk/
